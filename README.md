# Streamlit Demo App

Welcome to the **Streamlit Demo App**! This is a beginner-friendly app to explore the capabilities of Streamlit including charts, media embedding, file upload, interactive widgets, and forms.

---

## Features

1. **Greeting Section**
   - Enter your name and click "Greet" to display a personalized message.

2. **Random Data Charts**
   - Line chart and bar chart using random data generated with NumPy and Pandas.

3. **Sidebar**
   - Displays a title, image, and additional information.

4. **Media Embedding**
   - Supports images and YouTube videos.

5. **File Upload**
   - Upload CSV files and view their contents in a table.

6. **Text and Markdown Demonstration**
   - Showcases headings, subheadings, markdown, code snippets, and more.

7. **Interactive Widgets**
   - Text input, text area, number input, slider, selectbox, multiselect, radio buttons, checkboxes.

8. **Form**
   - A simple login form with username and password fields.
